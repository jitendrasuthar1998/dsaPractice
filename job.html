Job Description
      We are looking for a Frontend Engineer with strong experience in building scalable, responsive and cross-browser compatible web applications using ReactJS, Redux and other related libraries. To succeed in this role, you should be focused on building and creating a better end-user experience.
      Responsibilities
  • Build scalable web applications using ReactJS, Redux and other related libraries.
  • Convert visual designs to UI elements with an acute eye for detail and design fidelity.
  • Make web applications responsive and cross-browser compatible.
  • Optimize application performance and reduce page loading time.
  • Collaborate with various teams to develop new features.
  • Ability to work under pressure and deliver projects with tight deadlines.
      • Requirements
  • 2+ years of experience in front-end development with expertise in React.js.
  • Experience in Redux, Webpack with ES6 syntax.
  • Must have in-depth knowledge of JavaScript and DOM.
  • Experience in building Single-Page Applications.
  • Knowledge of common security vulnerabilities in web applications.
  • Experience in working with RESTful web services is a plus.
  • Preferred experience in developing responsive web applications using Sass and Bootstrap.
  • Ability to create well-abstracted, reusable UI components.
  • A keen eye for producing clean and semantic markup.
  • Capable of working independently in fast-paced environment.
